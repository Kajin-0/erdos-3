# Reserve-pattern product factorization

## Status

Symbolic factorization theorem for repeated backbone-middle latent demands generated by a common affine layer pattern.

It identifies the exact scalar defect exposed by the rank-two reserve counterexample. It does not yet prove that the corresponding parent three-AP edge occurrence is globally available after all other payments.

---

## 1. Layer graph

Let `R` be a finite set of layer offsets and let

```math
H=(R,\mathcal A)
```

be a finite graph whose action edge

```math
(c,o)\in\mathcal A
```

comes from an oriented selected three-AP layer

```math
s,c,o,
\qquad c-s=o-c.
```

For every connected component `K` of `H`, define its matching defect

```math
\delta(K)=\max(0,|E(K)|-|V(K)|)
          =\max(0,\beta(K)-1).
```

Put

```math
\delta(H)=\sum_K\delta(K).
```

This is the exact number of layer demands that cannot be assigned injectively to one of their center/opposite endpoints.

---

## 2. Common root-pair family

Let `Q` be a finite root set and let

```math
\mathcal E\subseteq\binom Q2
```

be a physical root-pair family.

For a pair

```math
e=\{u,v\}
```

and layer offset `r`, write

```math
e+r=\{u+r,v+r\}.
```

Assume that for every action layer `(s,c,o)` and every `e in E`:

```text
e+s is a duplicated latent demand;
e+c is its center reserve;
e+o is its opposite reserve.
```

Assume also that the physical translation map

```math
(e,r)\longmapsto e+r
```

is injective on

```math
\mathcal E\times V(H).
```

This injectivity separates the physical reserve graph into one copy of `H` for every root pair `e`.

## 3. Exact factorization

Fix

```math
e=\{u,v\},
\qquad g_e=v-u.
```

Every demand and reserve vertex in the copy `H_e` has pair weight

```math
\frac1{g_e}.
```

Therefore one component `K` contributes exact unmatched mass

```math
\frac{\delta(K)}{g_e}.
```

Summing first over components and then over root pairs gives

```math
\begin{aligned}
D_{\rm reserve}(H,\mathcal E)
&=
\sum_{e\in\mathcal E}
\sum_{K\subseteq H}
\frac{\delta(K)}{g_e}\\
&=
\delta(H)
\sum_{e\in\mathcal E}\frac1{g_e}.
\end{aligned}
```

Hence

```math
\boxed{
D_{\rm reserve}(H,\mathcal E)
=
\delta(H)J(\mathcal E).
}
```

This is an identity, not an inequality.

## 4. Three-AP coordinate state

Suppose

```math
Q=\{a,a+h,a+2h\}
```

and

```math
\mathcal E=\binom Q2.
```

Then

```math
J(\mathcal E)
=
\frac1h+rac1h+rac1{2h}
=
\frac5{2h}.
```

Therefore

```math
\boxed{
D_{\rm reserve}(H,Q)
=
\delta(H)\frac5{2h}.
}
```

One unit of layer matching defect is exactly one complete three-edge occurrence of the coordinate three-AP.

## 5. Rank-two no-go example

In `docs/lexicographic-reserve-rank-two-no-go.md`, the layer reserve graph on offsets

```math
\{0,1,5,6\}
```

has edges

```text
0--1,
5--6,
1--5,
1--6,
6--0.
```

It is `K_4` with one edge removed, so

```math
|E|=5,
\qquad |V|=4,
\qquad \beta=2,
\qquad \delta(H)=1.
```

The coordinate root set is

```math
Q=\{12000,12050,12100\},
```

with step `h=50`. Thus

```math
D_{\rm reserve}
=
1\cdot\frac5{2\cdot50}
=
\boxed{\frac1{20}},
```

exactly as certified by the independent verifier.

## 6. Production interface

If the coordinate state `Q` is retained as one named parent three-AP occurrence, its complete full-edge token family has mass

```math
\frac5{2h}.
```

Consequently a product pattern with

```math
\delta(H)=1
```

can be paid exactly by one occurrence of the coordinate witness, provided that occurrence has not already been consumed by:

```text
local activated-pair discharge;
current-latent terminal absorption;
an earlier reserve-pattern defect;
or another branching output.
```

This is the correct ownership question. The rank-two no-go does not create an unexplained scalar deficit; it converts the deficit into a named coordinate-state edge occurrence.

## 7. Remaining theorem

A whole-tree proof now needs a first-appearance rule for product-pattern defects:

```math
(H,Q)
\longmapsto
\text{one or more owned three-AP edge occurrences in }Q.
```

The required statement must control:

1. reuse of the same coordinate three-AP by several layer patterns;
2. patterns with `delta(H)>1`;
3. noninjective physical translations `(e,r) -> e+r`;
4. coordinate sets larger than one three-AP;
5. compatibility with the existing production-token partition.

The product identity determines the exact amount to be paid. Only occurrence ownership remains unresolved.
