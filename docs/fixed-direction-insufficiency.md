# Fixed-direction insufficiency

## Status

Proof-audit warning.  This note checks a possible false route in the quadratic-level pair-fiber program.

A first-order chord-graph spectral gap is not enough.  Even a second-order statement for each fixed direction separately is not enough.  The useful leverage must come from the fact that all pair-fibers are generated by one common set `B`.

## Setup

Let

```math
V_t=\{x:Q(x)=t\}
```

be a high-rank quadratic level set.  For an isotropic direction `d`, define

```math
X_d=V_t\cap\{x:B_Q(x,d)=0\}
```

in the homogeneous normalized model, and

```math
P_d=\{x\in X_d:x\in B,\ x+d\in B\}.
```

If `B` is internally 4AP-free, then

```math
P_d\cap(P_d-2d)=\emptyset.
```

## Why fixed-direction mixing cannot suffice

For fixed `d`, the slice `X_d` is a union of full internal lines parallel to `d`.  On each such line, the shift

```math
T_d:x\mapsto x+2d
```

is just a permutation of a cycle of length `p`.

The condition

```math
P_d\cap T_d^{-1}P_d=\emptyset
```

only says that `P_d` contains no two points at a prescribed offset on each `p`-cycle.

For small density this is easy.  Since the expected pair-fiber density is

```math
|P_d|/|X_d|\approx \beta^2,
```

and in the target regime

```math
\beta\sim n^{-1-\epsilon},
```

we have

```math
\beta^2\ll 1.
```

A subset of a `p`-cycle of density much smaller than a fixed positive constant can avoid a fixed shift without contradiction.  Thus no argument that treats each `P_d` as an arbitrary subset of `X_d` can prove the needed bound.

## What structure was lost

The missing constraint is that

```math
P_d(x)=1_B(x)1_B(x+d)1_{X_d}(x).
```

The family `P_d` is not arbitrary across directions.  All pair-fibers are generated by the same vertex set `B`.

This is the same overdetermination that appeared in the global pair-fiber branch:

```math
B_d=A\cap(A-d),
\qquad
B_d\cap(B_d-2d)=\emptyset.
```

Individual disjointness is weak.  Joint generation by one set is the only possible source of exponent gain.

## Correct second-order object

The relevant object is not a single shift system

```math
(X_d,T_d,P_d).
```

It is the incidence tensor

```math
\mathcal P(x,d)=1_B(x)1_B(x+d)1_{Q(d)=0}1_{B_Q(x,d)=0}.
```

Internal 4AP-freeness is

```math
\mathcal P(x,d)\mathcal P(x+2d,d)=0.
```

But the useful constraints are the coupled marginals:

```math
E_d\mathcal P(x,d)\approx \beta 1_B(x)
```

from chord-graph mixing, and

```math
E_x\mathcal P(x,d)\approx \beta^2 1_{Q(d)=0}
```

from direction-pair-density flatness.

The hard branch is therefore an overdetermined tensor-packing problem:

```math
\mathcal P\text{ has random-scale marginals but zero self-overlap along }(x,d)\mapsto(x+2d,d).
```

## Candidate tensor-packing lemma

A plausible theorem is:

> Let `V_t` be high-rank and let `B subset V_t` have no low-rank density increment.  Suppose the incidence tensor
>
> ```math
> \mathcal P(x,d)=1_B(x)1_B(x+d)1_{Q(d)=0}1_{B_Q(x,d)=0}
> ```
>
> has random-scale start and direction marginals:
>
> ```math
> E_d\mathcal P(x,d)\approx \beta 1_B(x),
> \qquad
> E_x\mathcal P(x,d)\approx \beta^2 1_{Q(d)=0}.
> ```
>
> If
>
> ```math
> \mathcal P(x,d)\mathcal P(x+2d,d)=0
> ```
>
> for all nonzero admissible `d`, then either `B` has a low-rank/quadratic density increment or
>
> ```math
> \beta\le C_p n^{-1-\epsilon_h}.
> ```

## Why this is sharper

This formulation avoids the two known losses:

1. compressing the obstruction to a single relative `U^3` norm, which gives only `beta^4`;
2. treating each direction independently, where exact shifted disjointness is too weak at density `beta^2`.

The proof must use both marginals and the common origin of all pair-fibers from `B`.

## Next research question

Can the tensor-packing lemma be attacked by a Cauchy--Schwarz or entropy argument using both marginals before collapsing to relative `U^3`?  If every such argument collapses back to `beta^4`, the route requires a genuinely new directional sifting or algebraic-expansion input.
