## 1. Direct rectangle radius

For a finite base state `B`, define the direct rectangle-support set

```math
\mathcal R(B)
=
\{U>0:\mathcal F_B(U,-U;0)>0\}.
```

Its contiguous support radius is

```math
\nu(B)
=
\max\{U\ge0:[1,U]\subseteq\mathcal R(B)\}.
```

Suppose the current state has the replicated form

```math
G_S(B)=B\cup(B+S)\cup(B+2S).
```

The four-ratio transport theorem gives obstruction at each separation

```math
T=kS\pm u,
\qquad
k\in\{1,2,3,4\},
\qquad
1\le u\le\nu(B).
```

The boundary points `T=kS` are obstructed by pure layer-index progressions. Therefore the complete interval support generated by the contiguous radius `U=nu(B)` is

```math
\mathcal Q(S,U)
=
\bigcup_{k=1}^4
[kS-U,kS+U]\cap\mathbb Z.
```

Every sponsor-compatible, layer-disjoint candidate in `mathcal Q(S,U)` is blocked.

## 2. Exact integer coalescence threshold

Consecutive windows

```math
[kS-U,kS+U]
```

and

```math
[(k+1)S-U,(k+1)S+U]
```

have no missing integer between them exactly when

```math
kS+U+1\ge(k+1)S-U.
```

Equivalently,

```math
\boxed{2U+1\ge S.}
```

When this condition holds, all four transport windows coalesce into the single integer interval

```math
\boxed{
\mathcal Q(S,U)
=
[S-U,4S+U]\cap\mathbb Z.
}
```

For odd `S`, the minimum coalescing radius is

```math
U_{\rm overlap}=\frac{S-1}{2}.
```

This is the exact integer version of the heuristic threshold `U/S >= 1/2`.

## 3. Radius demand of a target interval

Let

```math
I=[A,B]\cap\mathbb Z
```

be the admissible interval that must be blocked. Define its four-center transport demand by

```math
q_S(I)
=
\max_{T\in I}
\min_{1\le k\le4}|T-kS|.
```

Then

```math
\boxed{
I\subseteq\mathcal Q(S,U)
\quad\Longleftrightarrow\quad
U\ge q_S(I).
}
```

The function

```math
T\mapsto\min_{1\le k\le4}|T-kS|
```

is piecewise linear on the integer line. Its maximum over `I` occurs at:

1. an endpoint of `I`; or
2. an integer adjacent to a midpoint between consecutive centers `kS` and `(k+1)S`.

Thus `q_S(I)` is computable from a constant-size exact test; no scan over the interval is required.

Define the closure margin

```math
\boxed{
m_S(B;I)=\nu(B)-q_S(I).
}
```

Then:

```text
m_S(B;I) >= 0  -> the contiguous rectangle radius closes the complete target interval;
m_S(B;I) < 0   -> at least one target separation lies outside the four transport windows.
```

## 7. Scope

This note proves only the interval geometry and the exact `S_10` radius margin. It does not prove:

1. monotonic growth of `nu(B)` under arbitrary replication;
2. contraction of support holes;
