# Off-diagonal correction and direction-energy transfer

## Status

Substantive correction and new lemma in the high-`J` branch.

Two conclusions are recorded:

1. the negative centered interaction obtained by fixed-size direction sampling is largely the deterministic missing diagonal and does not amplify genuine off-diagonal discrepancy;
2. a dense graph of large skew interactions `J(d,e)` forces additive energy in a lower-threshold popular-direction set.

The second statement is a genuine structural consequence.  Its transfer back to a density increment in `A` has a quantitative threshold loss, identified below.

## Setup

Let `G` be a finite abelian group of size

```math
N=|G|,
```

and let `A subset G` be 4-AP-free.

For a signed direction `r`, define

```math
P_r=\{x:x,x+r\in A\},
\qquad
m(r)=|P_r|.
```

For directions `d,e`, define

```math
J(d,e)
=
|\{x:x,x+d,x+2e,x+3e\in A\}|.
```

The 4-AP-free condition gives

```math
J(d,d)=0
```

for every nonzero `d`.

Let `D` be a finite direction set and, for `E subset D`, write

```math
M(E)=\sum_{d\in E}m(d).
```

## Raw centered interaction

Define

```math
\Delta(E)
=
\sum_{d,e\in E}J(d,e)
-
\frac{M(E)^2}{N}.
```

Expanding the independence baseline into diagonal and off-diagonal pieces gives

```math
\Delta(E)
=
\sum_{\substack{d,e\in E\\d\ne e}}
\left(J(d,e)-\frac{m(d)m(e)}{N}\right)
-
\frac1N\sum_{d\in E}m(d)^2.
```

Therefore the genuinely off-diagonal discrepancy is

```math
\Gamma(E)
=
\sum_{\substack{d,e\in E\\d\ne e}}
\left(J(d,e)-\frac{m(d)m(e)}{N}\right),
```

and the exact relation is

```math
\boxed{
\Gamma(E)
=
\Delta(E)+\frac1N\sum_{d\in E}m(d)^2.
}
```

Thus `Delta(E)<0` does not by itself imply off-diagonal depletion.

## Pseudorandom calibration

Suppose the off-diagonal entries exactly match the independent model:

```math
J(d,e)=\frac{m(d)m(e)}{N}
\qquad(d\ne e).
```

Then

```math
\Gamma(E)=0,
```

but

```math
\Delta(E)
=-\frac1N\sum_{d\in E}m(d)^2<0.
```

For balanced pair counts, the relative size of this deficit is approximately `1/|E|`.  This is precisely the diagonal effect seen in the fixed-size sampling identity.

## Fixed-size sampling does not amplify Gamma

Let `|D|=K`, and choose `E` uniformly among all `s`-element subsets of `D`.

Every ordered pair `d ne e` is included with probability

```math
b=\frac{s(s-1)}{K(K-1)}.
```

Hence

```math
\boxed{
\mathbb E\Gamma(E)=b\Gamma(D).
}
```

The sign and normalized scale of genuine off-diagonal discrepancy are unchanged.  Fixed-size sampling makes the missing diagonal visible in `Delta`; it does not create a new off-diagonal obstruction.

This corrects the earlier interpretation of the centered-sampling branch.

# Large skew interactions create popular differences

For `tau>0`, define the popular-direction set

```math
D_\tau=\{r\in G:m(r)\ge\tau\}.
```

Define the large-interaction graph

```math
\mathcal G_\tau
=
\{(d,e)\in D\times D:J(d,e)\ge\tau\}.
```

## Edge-to-difference lemma

If `(d,e) in mathcal G_tau`, then

```math
2e-d\in D_\tau
\qquad\text{and}\qquad
3e-d\in D_\tau.
```

### Proof

Each point counted by `J(d,e)` gives

```math
x+d,\ x+2e\in A,
```

whose difference is `2e-d`.  Distinct `x` give distinct first endpoints `x+d`, so

```math
m(2e-d)\ge J(d,e)\ge\tau.
```

Likewise, the pair

```math
x+d,\ x+3e\in A
```

has difference `3e-d`, and therefore

```math
m(3e-d)\ge J(d,e)\ge\tau.
```

This proves the claim.

Moreover,

```math
(3e-d)-(2e-d)=e.
```

Thus every large-`J` edge gives a representation of `e` as a difference of two elements of `D_tau`.

## Multiplicity preservation

For `e in D`, define the indegree

```math
q(e)=|\{d\in D:(d,e)\in\mathcal G_\tau\}|.
```

For fixed `e`, the map

```math
d\mapsto 2e-d
```

is injective.  Hence the representations generated by distinct incoming edges are distinct, and

```math
\boxed{
r_{D_\tau-D_\tau}(e)\ge q(e).
}
```

Here

```math
r_{D_\tau-D_\tau}(h)
=|\{a\in D_\tau:a+h\in D_\tau\}|.
```

# Direction-energy lemma

The additive energy of `D_tau` is

```math
E(D_\tau)
=
\sum_h r_{D_\tau-D_\tau}(h)^2.
```

Using the multiplicity bound,

```math
E(D_\tau)
\ge
\sum_{e\in D}q(e)^2.
```

Let

```math
K=|D|,
\qquad
m=|\mathcal G_\tau|=\sum_e q(e).
```

Cauchy--Schwarz gives

```math
\sum_e q(e)^2\ge\frac{m^2}{K}.
```

Therefore

```math
\boxed{
E(D_\tau)
\ge
\frac{|\mathcal G_\tau|^2}{|D|}.
}
```

If the graph has density

```math
|\mathcal G_\tau|=\delta K^2,
```

then

```math
\boxed{
E(D_\tau)\ge\delta^2K^3.
}
```

This is a genuine additive-combinatorial consequence of the skew four-point configurations.

# Structure-or-expansion dichotomy

Let

```math
H=|D_\tau|.
```

Then

```math
\frac{E(D_\tau)}{H^3}
\ge
\delta^2\left(\frac{K}{H}\right)^3.
```

## Controlled-growth branch

If

```math
H\le LK,
```

then

```math
\frac{E(D_\tau)}{H^3}
\ge
\frac{\delta^2}{L^3}.
```

This is at the natural BSG scale.  A quantitative BSG theorem yields a substantial subset

```math
D'\subseteq D_\tau
```

with small doubling, with constants depending on `delta` and `L`.

## Direction-expansion branch

If

```math
H\gg K,
```

then the large-`J` graph has generated many new directions with pair count at least `tau`.

The total pair-mass identity

```math
\sum_{r\in G}m(r)=|A|^2
```

implies the universal bound

```math
|D_\tau|\le\frac{|A|^2}{\tau}.
```

Thus repeated direction expansion at a fixed threshold cannot continue indefinitely.

# Transfer from structured directions back to A

Suppose the controlled-growth branch gives

```math
D'\subseteq D_\tau.
```

Since every `r in D'` satisfies `m(r)>=tau`,

```math
\sum_{r\in D'}m(r)\ge\tau|D'|.
```

On the other hand,

```math
\sum_{r\in D'}m(r)
=
\sum_{x\in A}|(A-x)\cap D'|.
```

Therefore some `x in A` satisfies

```math
|(A-x)\cap D'|
\ge
\frac{\tau|D'|}{|A|}.
```

Equivalently,

```math
\boxed{
|A\cap(x+D')|
\ge
\frac{\tau}{|A|}|D'|.
}
```

So `A` has relative density at least

```math
\frac{\tau}{|A|}
```

inside a translate of the structured direction set `D'`.

## Threshold obstruction

If the ambient density is

```math
\alpha=\frac{|A|}{N},
```

then this becomes a genuine density increment only when

```math
\frac{\tau}{|A|}>\alpha,
```

or equivalently

```math
\boxed{
\tau>\frac{|A|^2}{N}=\alpha^2N.
}
```

But a pseudorandom skew interaction has scale

```math
J(d,e)\approx\alpha^4N.
```

Therefore a large-`J` graph at the natural skew scale only generates directions popular at threshold `alpha^4 N`, far below the pair baseline `alpha^2 N`.  The direct transfer then gives local density only about

```math
\frac{\alpha^4N}{\alpha N}=\alpha^3,
```

which is smaller than the ambient density `alpha`.

This is the current quantitative bottleneck.

# Revised high-J route

The valid structural chain is

```math
\text{dense graph of }J(d,e)\ge\tau
\Longrightarrow
E(D_\tau)\ge\frac{|\mathcal G_\tau|^2}{|D|}
```

and hence either

```math
\text{small-doubling structure in }D_\tau
```

or

```math
\text{expansion to many threshold-}\tau\text{ directions}.
```

However, transferring a structured subset of `D_tau` to a density increment in `A` requires

```math
\tau>\alpha^2N,
```

whereas the natural skew scale is `alpha^4N`.

The next proof task is therefore not merely to obtain small doubling.  It is to recover the missing factor `alpha^{-2}` by one of:

1. localization to an ambient set of density larger than `alpha` before applying the transfer;
2. a higher-multiplicity incidence argument using many common basepoints rather than only pair popularity;
3. an iteration in threshold that converts direction expansion into concentration before pair mass is exhausted.
