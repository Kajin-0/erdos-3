# Main proof target realignment after finite-state audit

## Status

Research-program control note.  This note keeps the project pointed at the full fixed-`k=4` Erdős reciprocal-sum problem while preserving the finite-state paper as a side deliverable.

## Main theorem target

For fixed `k=4`, the decisive target is dyadic summability:

```math
\sum_{j\ge 1} r_4(2^j)/2^j < \infty.
```

A sufficient concrete bound is

```math
r_4(N) \ll N/(\log N)^{1+\epsilon}
```

for some `epsilon>0`.

More generally, it is enough to prove any dyadically summable estimate, for example

```math
r_4(N)/N \ll 1/((\log N)(\log\log N)^{1+\epsilon}).
```

## Why this is the correct target

The dyadic summability equivalence shows that for fixed `k>=4`, the reciprocal-sum problem is equivalent to the above summability condition.

In particular, cross-block arithmetic constraints cannot solve the unrestricted problem if the dyadic extremal series diverges: one can isolate extremal blocks on one residue class of dyadic scales, preserve divergent reciprocal mass, and eliminate cross-block APs.

Thus the full problem has been reduced back to a quantitative extremal question for `r_4(N)`.

## Current known upper-bound regime

The current usable general-purpose upper-bound regime remains polylogarithmic:

```math
r_4(N) \ll N(\log N)^{-c}
```

for some positive constant `c`.

This is not enough unless `c>1`, and existing methods do not presently supply that threshold.

Therefore the main proof program needs either:

1. an improvement of the logarithmic exponent past `1`;
2. a different estimate with dyadically summable decay;
3. a new density-increment or recurrence mechanism not captured by current Green--Tao-style cost structure.

## What the finite-state program contributes

The finite-state / automatic-set result is useful but auxiliary.

It shows that no fixed regular digit language can be a divergent reciprocal-sum AP-free counterexample.  This eliminates a natural structured search class and gives a publishable computational-extremal side paper.

However, it does not address the unrestricted fixed-`k=4` problem, because unrestricted extremal dyadic blocks need not have bounded finite-state complexity.

## Main proof route retained

The central proof work should continue on the quantitative `r_4` side.  The current branch has already identified the following obstruction chain:

1. A 4AP-free set of density `alpha` has a signed 4AP deficit of size about `alpha^4`.
2. The deficit splits into either a negative trilinear branch or a pure four-balanced `U^3` branch.
3. The trilinear branch gives only an `alpha^2` hyperplane increment, exactly the logarithmic-barrier scale.
4. The pure `U^3` branch, under generic inverse machinery, is no better and usually worse.
5. Product tensor obstructions exist, but are removed by minimality because they have large fiber increments.
6. A minimal critical obstruction must be hyperplane-flat and genuinely higher-order.

This identifies the needed breakthrough:

```math
\text{4AP-free} + \text{signed deficit} + \text{minimal flatness}
```

must imply an increment or recurrence gain stronger than the `alpha^2` logarithmic-barrier scale.

## Next mathematical target

The next precise target is a strengthened one-shot recurrence/density-increment theorem:

> If `A subset [N]` or `A subset F_p^n` has density `alpha`, is 4AP-free, and is minimal/flat against all linear density increments at scale larger than the critical threshold, then the signed 4AP deficit forces either:
>
> 1. a density increment of size `alpha^{2-epsilon}` at acceptable codimension/cost; or
> 2. a direct contradiction by counting 4APs inside the structured object generated by the pure `U^3` branch.

This is the point where the full problem would move beyond existing logarithmic-barrier methods.

## Practical work split

### Main proof track

Continue attacking the minimal-flat signed-deficit problem and the pure `U^3` rank split:

- low-rank quadratic branch: track rank cost versus increment size;
- high-rank quadratic branch: seek a relative 4AP counting lemma inside high-rank quadratic level sets;
- trilinear branch: look for a way to upgrade the borderline `alpha^2` increment to `alpha^{2-epsilon}` using sign, boundedness, or no-4AP constraints.

### Side paper track

Develop the finite-state obstruction paper:

- automatic-set no-go theorem;
- DFA carry-product AP decision theorem;
- spectral growth scorer;
- Walker benchmark certificates;
- PB/MaxSAT benchmark pack.

The side track should not distract from the main proof target, but it can produce a defensible intermediate publication.
