# Affine-tree multiplicity lower bound

## Status

Exact recursion theorem for the concentrated three-AP extension branch.

The hereditary three-dilate class does not yield a density increment along a single recursive path.  In fact, every child has density at most that of its parent.  Nevertheless, the total normalized mass of all children nearly doubles above the Roth-error scale.  Persistent recursion therefore forces exponentially many affine paths to terminate at the same small-scale direction.

This converts the previous qualitative overlap bottleneck into a precise fixed-leaf multiplicity problem.

## Hereditary class and child fibers

Let

```math
D\subseteq[1,M]
```

satisfy

```math
D\cup2D\cup3D
\text{ is 4-AP-free}.
```

For an integer `q`, define

```math
E_q(D)
=
\{d:q+d,q+2d,q+3d\in D\}.
```

Every nonempty child lies in `[1,M/2]` and again satisfies

```math
E_q(D)\cup2E_q(D)\cup3E_q(D)
\text{ is 4-AP-free}.
```

## No density increment along one edge

Put

```math
E=E_q(D).
```

The two sets

```math
q+E
\qquad\text{and}\qquad
q+2E
```

are disjoint subsets of `D`.

Indeed, if

```math
q+e_1=q+2e_2
```

with `e_1,e_2 in E`, then `e_1=2e_2`, contradicting

```math
E\cap2E=\varnothing.
```

Therefore

```math
2|E|\le|D|.
```

Writing

```math
\beta(D)=\frac{|D|}{M},
\qquad
\beta(E)=\frac{|E|}{M/2},
```

we obtain

```math
\boxed{
\beta(E)\le\beta(D).
}
```

Thus recursive concentration never increases density at its natural halved scale.

## Binary disjoint-copy lemma

Consider a depth-`h` path

```math
D_0=D,
\qquad
D_t=E_{q_t}(D_{t-1})
\quad(1\le t\le h).
```

For

```math
\varepsilon=(\varepsilon_1,\dots,\varepsilon_h)
\in\{1,2\}^h,
```

define the affine map

```math
\Phi_\varepsilon(x)
=
q_1
+\varepsilon_1q_2
+\varepsilon_1\varepsilon_2q_3
+\cdots
+\left(\prod_{t=1}^{h-1}\varepsilon_t\right)q_h
+\left(\prod_{t=1}^{h}\varepsilon_t\right)x.
```

Then the `2^h` sets

```math
\Phi_\varepsilon(D_h),
\qquad
\varepsilon\in\{1,2\}^h,
```

are pairwise disjoint subsets of `D_0`.

### Proof

At one generation, the two images

```math
q_1+D_1,
\qquad
q_1+2D_1
```

are disjoint subsets of `D_0`.

Assume the claim at depth `h-1` inside `D_1`.  Composing those disjoint images with either

```math
x\mapsto q_1+x
```

or

```math
x\mapsto q_1+2x
```

preserves disjointness.  Images whose first digit is `1` lie inside `q_1+D_1`, while images whose first digit is `2` lie inside `q_1+2D_1`; those two parent images are disjoint.  Induction proves the claim.

Consequently,

```math
\boxed{
2^h|D_h|\le|D_0|.
}
```

Since `D_h subseteq [1,M/2^h]`, this is equivalent to

```math
\boxed{
\frac{|D_h|}{M/2^h}
\le
\frac{|D_0|}{M}.
}
```

Again, no descendant can have larger natural-scale density than the root.

## Total normalized child mass

Let

```math
T_3(D)=\sum_q|E_q(D)|.
```

The deletion argument gives

```math
T_3(D)\ge|D|-r_3(M).
```

Therefore

```math
\sum_q\beta(E_q(D))
=
\sum_q\frac{|E_q(D)|}{M/2}
=
\frac{2T_3(D)}M
\ge
2\left(
\beta(D)-\frac{r_3(M)}M
\right).
```

Thus every child density is at most the parent density, but the sum of all child densities is nearly twice the parent density.

This is a forced-branching phenomenon rather than a density increment.

## Recursive tree notation

Let `mathcal T_h` be the set of depth-`h` predecessor paths.  For a node `u` at depth `t`, let

```math
D_u\subseteq[1,M/2^t]
```

be its fiber and put

```math
\beta_u
=
\frac{|D_u|}{M/2^t}.
```

Define the total normalized mass at depth `t` by

```math
S_t
=
\sum_{|u|=t}\beta_u.
```

Suppose that for every node of depth less than `h`,

```math
\frac{r_3(M/2^{|u|})}{M/2^{|u|}}
\le
\eta\beta_u
```

for some `0<=eta<1`.

Then every node satisfies

```math
\sum_{v\text{ child of }u}\beta_v
\ge
2(1-\eta)\beta_u.
```

Summing over each generation gives

```math
\boxed{
S_h
\ge
[2(1-\eta)]^h\beta_0.
}
```

## Forced fixed-leaf multiplicity

For a direction

```math
d\in[1,M/2^h],
```

define its depth-`h` path multiplicity by

```math
m_h(d)
=
|\{u\in\mathcal T_h:d\in D_u\}|.
```

Double counting leaf incidences gives

```math
\sum_d m_h(d)
=
\sum_{|u|=h}|D_u|
=
\frac{M}{2^h}S_h.
```

There are at most `M/2^h` possible directions `d`.  Therefore

```math
\max_d m_h(d)
\ge
S_h.
```

Combining with the recursive lower bound yields

```math
\boxed{
\max_d m_h(d)
\ge
[2(1-\eta)]^h\beta_0.
}
```

Thus persistent low-error recursion forces one small-scale direction to occur along exponentially many distinct affine predecessor paths.

## Affine form generated by one path

A depth-`h` path

```math
(q_1,\dots,q_h)
```

ending at direction `d` implies that the root set contains every form

```math
q_1
+i_1q_2
+i_1i_2q_3
+\cdots
+\left(\prod_{t=1}^{h-1}i_t\right)q_h
+\left(\prod_{t=1}^{h}i_t\right)d,
```

where

```math
i_1,\dots,i_h\in\{1,2,3\}.
```

The full family may contain collisions.  However, the binary subfamily with

```math
i_t\in\{1,2\}
```

contains exactly `2^h` distinct points, by the binary disjoint-copy lemma.

Hence each path ending at `d` gives a binary affine cube of size `2^h` inside the root.

## Cube-overlap identity

Fix `d`.  For a path `pi` ending at `d`, let

```math
C_\pi(d)
```

be its binary affine cube of `2^h` root points.  Define

```math
c_h(x,d)
=
|\{\pi:x\in C_\pi(d)\}|.
```

Then

```math
\boxed{
2^hm_h(d)
=
\sum_{x\in D_0}c_h(x,d).
}
```

Therefore

```math
\boxed{
m_h(d)
\le
\frac{|D_0|}{2^h}
\max_{x\in D_0}c_h(x,d).
}
```

The path-multiplicity problem is thus equivalent to controlling overlap among binary affine cubes with the same terminal direction.

## Depth-one calibration

For fixed `d`, the depth-one predecessors are the integers `q` such that

```math
q+d,q+2d,q+3d\in D_0.
```

The corresponding three-point progressions are pairwise disjoint.  Indeed, two starts differing by `d`, `2d`, or `3d` would place four consecutive points of a `d`-progression inside `D_0`, creating a 4-AP.  Starts farther apart give disjoint triples.

Hence

```math
\boxed{
m_1(d)\le\frac{|D_0|}{3}.}
```

This is not yet strong enough for iteration, but it shows that fixed-leaf multiplicity is constrained by 4-AP-freeness even at the first generation.

## Revised bottleneck

The recursion now gives an exact lower bound

```math
m_h(d)
\gtrsim
2^h\beta_0
```

for some terminal direction `d`, provided the Roth errors remain proportionally small.

To close the argument, one needs an upper bound for fixed-terminal affine-path multiplicity.  Any estimate of the form

```math
m_h(d)
\le
C^h\mathrm{poly}(M)
```

with an effective growth rate below the forced branching rate, or an overlap estimate

```math
\max_x c_h(x,d)=o(2^h),
```

in the relevant range, would convert persistent recursion into a contradiction.

The immediate target is therefore:

> Prove that a 4-AP-free three-dilate set cannot support too many depth-`h` predecessor paths with one common terminal direction.

This is now a concrete finite combinatorial statement, independent of harmonic weights, that would complete the concentrated cross-scale branch when combined with the linear extension-load lower bound.
