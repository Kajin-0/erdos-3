# Double-translate four-ratio exclusion

## Status

State-independent difference exclusion for repeated completion-step states.

The theorem identifies four rectangle aspect ratios that are impossible in any
four-AP-free double-translate configuration. It applies directly to the
state/reference rectangle ledger from repeated heavy completion fibers.

---

## 1. Double-translate product

Let

```math
P¥subseteq¥mathbb Z
```

be four-AP-free. Let `C` and `S` be finite sets and fix

```math
¥sigma¥in¥{-1,+1¥}.
```

Assume

```math
C+¥sigma S¥subseteq P,
¥qquad
C+2¥sigma S¥subseteq P.
```

Choose

```math
c_0<c_1¥in C,
¥qquad
¥delta=c_1-c_0>0,
```

and

```math
d_0<d_1¥in S,
¥qquad
r=d_1-d_0>0.
```

Reflection preserves arithmetic progressions, so it suffices to display the
right-oriented formulas.

---

## 2. Ratio `delta=2r`

The first-copy rectangle contains

```math
c_0+d_0,
¥quad
c_0+d_1,
¥quad
c_1+d_0,
¥quad
c_1+d_1.
```

When `delta=2r`, these are

```math
x,
¥quad
x+r,
¥quad
x+2r,
¥quad
x+3r.
```

Thus

```math
¥boxed{¥delta¥ne2r.}
```

---

## 3. Ratio `r=2delta`

Using the alternate rectangle order,

```math
c_0+d_0,
¥quad
c_1+d_0,
¥quad
c_0+d_1,
¥quad
c_1+d_1
```

has successive differences

```math
¥delta,
¥quad
r-¥delta,
¥quad
¥delta.
```

When `r=2delta`, this is a four-term progression. Therefore

```math
¥boxed{¥delta¥ne r/2.}
```

The ratio requires `r` even in the integer setting.

---

## 4. Ratio `delta=r`

The second-copy rectangle contains

```math
c_0+2d_0,
¥quad
c_1+2d_0,
¥quad
c_0+2d_1,
¥quad
c_1+2d_1.
```

When `delta=r`, these are spaced by `r`. Hence

```math
¥boxed{¥delta¥ne r.}
```

---

## 5. Ratio `delta=4r`

Using the other second-copy order,

```math
c_0+2d_0,
¥quad
c_0+2d_1,
¥quad
c_1+2d_0,
¥quad
c_1+2d_1
```

is spaced by `2r` when `delta=4r`. Thus

```math
¥boxed{¥delta¥ne4r.}
```

## 6. Difference-set form

Write

```math
¥Delta^+(X)=¥{y-x:x<y,¥ x,y¥in X¥}.
```

The four exclusions are equivalent to

```math
¥boxed{
¥Delta^+(C)
¥cap
¥left(
¥tfrac12¥Delta^+(S)
¥cup
¥Delta^+(S)
¥cup
2¥Delta^+(S)
¥cup
4¥Delta^+(S)
¥right)
=
¥varnothing,
}
```

where the half-scaled set contains only integer values.

Thus the reference-difference reserve avoids four exact dyadic aspect copies
of the internal step-difference set.

---

## 7. Rectangle-token consequence

For every repeated-state rectangle cell `(delta,d)`, the collision weight is
organized by the aspect ratio `delta/d`. The present theorem does not bound all
aspect bands, but it removes the four ratios generated by any internal pair
difference

```math
r=|d_1-d_0|.
```

The correct token therefore records both:

```text
reference difference delta;
internal pair difference r;
dyadic aspect band;
copy pattern producing the exclusion.
```

These are genuine structural zeroes in the two-scale Bellman matrix, not fitted
features.

---

## 8. Scope

The theorem excludes four exact ratios. It does not imply a uniform gap around
those ratios and does not by itself bound arbitrary far-aspect rectangles.

The next quantitative theorem must combine these zeroes with one of:

1. additive growth of the step-difference set;
2. first-appearance packing of aspect tokens;
3. dyadic band summation;
4. further mixed-copy obstruction classes.
