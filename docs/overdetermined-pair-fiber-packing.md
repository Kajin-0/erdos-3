# Overdetermined pair-fiber packing

## Status

Proof-audit formulation.  This note identifies the real hard object left after the pair-density variance dichotomy: a globally coupled family of nearly uniform pair-fibers, each disjoint from its own translate.

## Setup

Let

```math
G=F_p^n,
\qquad p>4,
```

and let

```math
A\subset G,
\qquad
\alpha=|A|/|G|,
\qquad
a=1_A.
```

For each direction `d`, define

```math
B_d=A\cap(A-d),
\qquad
b_d(x)=1_{B_d}(x)=a(x)a(x+d),
```

and

```math
p_d=E_x b_d(x).
```

The hard branch after the pair-density variance dichotomy is:

```math
p_d\approx \alpha^2
```

for most `d`, while for every nonzero `d`,

```math
B_d\cap(B_d-2d)=\emptyset.
```

Equivalently,

```math
b_d(x)b_d(x+2d)=0.
```

## Individual disjointness is too weak

For a fixed `d`, the condition

```math
B_d\cap(B_d-2d)=\emptyset
```

is weak at density

```math
|B_d|/|G|\approx \alpha^2.
```

A sparse set of density `alpha^2` can easily avoid one translate.  Therefore the power of the condition cannot come from any single `B_d` in isolation.

The only possible gain is from the fact that all fibers `B_d` are generated by the same set `A`.

## Coupling identity 1: start-incidence

A completely arbitrary family of sets of density `alpha^2` would satisfy

```math
E_d 1_{B_d}(x)\approx \alpha^2
```

for most `x`.

But the genuine pair-fiber family satisfies exactly

```math
E_d b_d(x)=a(x)E_d a(x+d)=\alpha a(x).
```

Thus the family `B_d` is not spread uniformly over `G`; its starts are concentrated exactly on `A`.

This is a strong global coupling:

```math
E_d b_d = \alpha 1_A.
```

Any sifting or almost-periodicity argument should exploit this identity.

## Coupling identity 2: pair-density Fourier flatness

The hard branch also has

```math
E_d |p_d-\alpha^2|^2\ll \alpha^{5-2\epsilon}.
```

Equivalently, the difference distribution of `A` is flat at the exponent-gain scale.  Therefore a proof cannot rely merely on uneven pair densities.

## Coupling identity 3: fiber overlaps encode corners

For two directions `d,e`,

```math
E_x b_d(x)b_e(x)
=E_x a(x)a(x+d)a(x+e).
```

Thus overlaps of pair-fibers encode 3-point corners with common basepoint.

More generally, shifted overlaps

```math
E_x b_d(x)b_e(x+h)
```

encode 3- and 4-point configurations involving the directions `d,e,h`.

Therefore the family `{B_d}` is highly constrained: its pairwise intersections are not arbitrary; they are lower-dimensional pattern counts of `A`.

## Coupling identity 4: exact forbidden self-overlap

For every nonzero `d`,

```math
E_x b_d(x)b_d(x+2d)=0.
```

This is a missing shifted self-overlap at one special shift depending on `d`.

In the uniform branch, a random-like pair-fiber of density `alpha^2` would have expected shifted self-overlap

```math
\alpha^4.
```

Thus each direction imposes a missing amount about `alpha^4`, but the missing shifts vary with `d`.

The challenge is to aggregate these missing self-overlaps across directions without collapsing back to the weak global `U^3` conclusion.

## Candidate overdetermined-packing lemma

A possible deterministic lemma is:

> Let `A subset F_p^n` be hyperplane-flat and suppose its pair densities are `L^2`-flat:
>
> ```math
> E_d |p_d-alpha^2|^2 \ll alpha^{5-2epsilon}.
> ```
>
> If, for every nonzero `d`, the genuine pair-fiber
> ```math
> B_d=A\cap(A-d)
> ```
> satisfies
> ```math
> B_d\cap(B_d-2d)=emptyset,
> ```
> then either `A` has an affine density increment of size `alpha^{2-epsilon'}` or the family `{B_d}` has high-rank quadratic organization.

This would bridge the physical-space/sifting branch to the existing low-rank/high-rank quadratic split.

## Why a line-fiber-only argument is unlikely to suffice

For each projective direction, the restrictions of `A` to affine lines in that direction are 4AP-free subsets of `F_p`.

However, when `alpha` is small and `p` is fixed, most line fibers are tiny.  Line-by-line finite constraints alone may be too weak to produce the required exponent gain.

The useful object is not one line or one direction.  It is the global packing of all pair-fibers across all directions, together with the start-incidence identity

```math
E_d b_d=\alpha 1_A.
```

## Almost-periodicity formulation

Define the pair-fiber averaging operator

```math
\mathcal B A(x,d)=a(x)a(x+d).
```

The hard branch says:

1. each fiber in `x` has density about `alpha^2` for most `d`;
2. the average over `d` recovers `alpha a(x)`;
3. each fiber misses the shift `2d` exactly;
4. pair-density variance is too small to expose a linear Fourier increment.

This is an overdetermined packing problem in `G x G`.  A physical-space proof should look for a set of translations or directions where this packing is not stable.  Failure of stability should be converted into a density increment before invoking a generic inverse theorem.

## Next research question

Can the identities

```math
E_d b_d=\alpha 1_A,
\qquad
E_x b_d(x)b_d(x+2d)=0,
\qquad
p_d\approx\alpha^2
```

hold simultaneously in a hyperplane-flat set at the critical scale

```math
n\sim \alpha^{-1+\delta}
```

without producing either a stronger-than-`alpha^2` increment or high-rank quadratic organization?
