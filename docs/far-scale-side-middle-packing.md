# Far-scale side-middle packing

## Status

Exact cardinality theorem for unmatched side-middle descendants in the coordinated role recursion.

The shell-stable paired-lift theorem handles common descendant mass.  The remaining issue is unmatched mass: a side descendant shell `S` and a middle descendant set `T` may be different, and one affine lift can overlap heavily with the other.

When the base step `q` is far from the scale of `S`, such overlap has a rigid reflection cost.  Every completely absorbed side source forces two new mirror points in the middle lift.  Combining this with the points left unabsorbed gives an optimized packing bound.

For equal cardinalities, the lower bound has leading constant `8/3`, matching the coordinated harmonic branching factor.

## Setup

Let

```math
S\subseteq[R,2R]
```

be a finite positive-integer set satisfying

```math
S\cap2S=\varnothing.
```

Let `T` be any finite positive-integer set and let `q>0`.

Define the translated side and middle lifts

```math
A(S)=\{0\}\cup S\cup2S
```

and

```math
M_q(T)=\{q\}\cup(q-T)\cup(q+T).
```

Write

```math
m=|S|,
\qquad
n=|T|.
```

Because `S cap 2S` is empty,

```math
|A(S)|=1+2m.
```

Because `T` is positive, the three pieces `{q}`, `q-T`, and `q+T` are pairwise disjoint, so

```math
|M_q(T)|=1+2n.
```

Assume one of the two separated-scale conditions:

```math
q>4R
```

or

```math
0<q<R/2.
```

## Completely absorbed side sources

Define

```math
E
=
\{s\in S:s\in M_q(T)\text{ and }2s\in M_q(T)\}
```

and put

```math
e=|E|.
```

Thus `s in E` means that both non-anchor side-lift points generated by `s` are already present in the middle lift.

## Large-base regime: `q>4R`

For every `s in S`,

```math
0<s<q,
\qquad
0<2s<q.
```

If `s in E`, there must be elements

```math
t_s=q-s\in T
```

and

```math
u_s=q-2s\in T.
```

The middle lift then also contains their reflected partners

```math
q+t_s=2q-s
```

and

```math
q+u_s=2q-2s.
```

These reflected points lie strictly above `4R`, while

```math
A(S)\subseteq[0,4R].
```

Hence they are outside the side lift.

They are pairwise distinct as `s` varies.  Each of the two maps

```math
s\mapsto2q-s,
\qquad
s\mapsto2q-2s
```

is injective.  A cross collision would give

```math
2q-s_1=2q-2s_2,
```

hence

```math
s_1=2s_2,
```

contrary to `S cap 2S = emptyset`.

Thus every completely absorbed source creates two distinct new middle-lift points outside `A(S)`.

The middle anchor `q` is also outside `A(S)`.  Therefore

```math
\boxed{
|A(S)\cup M_q(T)|
\ge
2m+2+2e.
}
```

## Small-base regime: `0<q<R/2`

Now every `s in S` satisfies

```math
s>q,
\qquad
2s>q.
```

If `s in E`, there are elements

```math
t_s=s-q\in T
```

and

```math
u_s=2s-q\in T.
```

The middle lift also contains the reflected points

```math
q-t_s=2q-s<0
```

and

```math
q-u_s=2q-2s<0.
```

They lie outside

```math
A(S)\subseteq[0,4R].
```

The same injectivity and cross-collision argument shows that these are `2e` distinct points.  The anchor `q` lies strictly between `0` and `R`, so it is outside `A(S)` as well.

Hence the identical estimate holds:

```math
\boxed{
|A(S)\cup M_q(T)|
\ge
2m+2+2e.
}
```

## Bound from unabsorbed sources

For every

```math
s\in S\setminus E,
```

at least one of the two distinct side points

```math
s,\qquad2s
```

is absent from `M_q(T)`.

All `2m` branch points in `S union 2S` are mutually distinct.  Therefore one may choose a different point of

```math
A(S)\setminus M_q(T)
```

for every unabsorbed source.  Thus

```math
|A(S)\setminus M_q(T)|\ge m-e.
```

Since `|M_q(T)|=1+2n`,

```math
\boxed{
|A(S)\cup M_q(T)|
\ge
1+m+2n-e.
}
```

This estimate does not use scale separation; it is valid whenever `S cap 2S` is empty.

## Optimized far-scale packing theorem

Combining the two estimates gives

```math
\boxed{
|A(S)\cup M_q(T)|
\ge
\max\{2m+2+2e,\ 1+m+2n-e\}.
}
```

The right side is valid for the actual integer `e`.  Minimizing it over the full interval

```math
0\le e\le m
```

produces a bound depending only on `m` and `n`.

### Regime 1: `2n\le m+1`

The two affine functions cross at or before `e=0`.  Hence

```math
\boxed{
|A(S)\cup M_q(T)|\ge2m+2.
}
```

### Regime 2: `m+1\le2n\le4m+1`

The crossing point

```math
e_*=\frac{2n-m-1}{3}
```

lies in `[0,m]`.  At this point both bounds equal

```math
\frac43(m+n+1).
```

Therefore

```math
\boxed{
|A(S)\cup M_q(T)|
\ge
\frac43(m+n+1).
}
```

### Regime 3: `2n\ge4m+1`

The crossing lies at or beyond `e=m`.  Hence

```math
\boxed{
|A(S)\cup M_q(T)|\ge2n+1.
}
```

Combining all three cases:

```math
\boxed{
|A(S)\cup M_q(T)|
\ge
\begin{cases}
2m+2,
&2n\le m+1,\\[4pt]
\dfrac43(m+n+1),
&m+1\le2n\le4m+1,\\[8pt]
2n+1,
&2n\ge4m+1.
\end{cases}
}
```

The theorem applies in either separated-scale regime

```math
q>4R
```

or

```math
0<q<R/2.
```

The reflected last-point side lift satisfies the same conclusion.

## Equal-cardinality consequence

When

```math
m=n,
```

the middle regime applies and gives

```math
\boxed{
|A(S)\cup M_q(T)|
\ge
\frac83m+\frac43.
}
```

Thus two far-scale side-middle descendants of comparable cardinality cannot pack into the parent at binary rate.  Their combined physical-space cost has leading factor `8/3`, exactly the harmonic branching constant obtained from coordinated valuation compression.

## Weighted shell interpretation

For

```math
S\subseteq[R,2R],
```

one has

```math
R H(S)\le m\le2R H(S).
```

If `T` is also resolved into a shell with comparable reciprocal scale and comparable cardinality, the middle-regime estimate translates into a corresponding `8/3`-type lower bound for the parent-point capacity consumed by the two descendant masses.

The exact conversion depends on the shell scale of `T`; the cardinality theorem is the clean invariant statement.

## Consequence for the recursion

The unmatched-overlap problem is now localized to the comparable-scale zone

```math
R/2\le q\le4R.
```

Outside this zone:

- low absorption leaves many side branch points genuinely separate from the middle lift;
- high absorption creates two reflected middle points for every absorbed side source;
- optimizing the two effects gives the piecewise packing theorem above.

Therefore any recursive overlap capable of defeating the `8/3` harmonic growth must repeatedly concentrate in side-middle interactions whose base step and descendant shell remain within a fixed multiplicative factor.

## Remaining task

Develop a comparable-scale theorem for

```math
R/2\le q\le4R.
```

In this regime all relevant reciprocals are comparable, so a cardinality packing estimate can be converted directly into a harmonic estimate.  The paired common-subset theorem supplies factor `3` on the shared part; the unresolved component is packing of the two unmatched step sets at comparable scale.
