# Triangle average not forced by shear

## Status

Proof-audit barrier.  This note tests whether the original shear-energy hypothesis can force the anchored triangle averages needed for pair-lift coboundary cleaning.

The conclusion is negative at the level of available information:

```math
\text{large shear energy}
\Rightarrow
\text{many biased individual pair-lines}
```

but it does not by itself imply

```math
\text{large anchored triangle averages}.
```

The latter is a higher-order coherence statistic across different pair-lines.

## Shear energy is linewise

Recall

```math
Q_{\mathrm{shear}}
=\sum_w\sum_u h_w(u)\overline{h_w(u+2w)},
\qquad
h_w(u)=c_uc_{w-u}.
```

Decomposing into lines parallel to `w`,

```math
H_{w,r}(t)=h_w(r+tw),
```

gives

```math
Q_{\mathrm{shear}}
=\sum_{w,r}\sum_t H_{w,r}(t)\overline{H_{w,r}(t+2)}.
```

Thus the shear obstruction is a sum of autocorrelations of individual line functions.

Fourier extraction from this identity gives line-mode bias for many individual `(w,r)`.  It does not compare phases between different lines.

## Anchored triangle averages are third-order

For an anchor `o`, the anchored triangle average is

```math
\mathcal T_o(a,b)=E_s H_{a,b}(s)H_{b,o}(s)H_{o,a}(s).
```

Fourier expansion gives

```math
\mathcal T_o(a,b)=
\sum_{m_1+m_2+m_3=0}
\widehat H_{a,b}(m_1)
\widehat H_{b,o}(m_2)
\widehat H_{o,a}(m_3).
```

This is not controlled by the individual quantities

```math
C_2(H_{a,b})=\sum_m |\widehat H_{a,b}(m)|^2 e_p(-2m).
```

The autocorrelation sees the spectral distribution of each line separately.  The triangle average sees additive convolution among spectra of three different lines.

## Abstract obstruction model

In an abstract model, choose many active pair-lines and let each line have a pure mode:

```math
H_e(t)=A_e e_p(\mu_e t),
```

where `e` denotes an oriented pair-line.

Then each individual line has maximal mode bias.  The fixed-line extraction is as strong as possible.

But for an anchored triangle `(a,b,o)`, the projected triangle average is nonzero only if

```math
\mu_{ab}+\mu_{bo}+\mu_{oa}=0.
```

If the labels `mu_e` are chosen independently subject only to reversal symmetry, then this closure condition holds with probability about `1/p`, not with positive density close to one.

Thus line-mode bias alone is compatible with almost no useful triangle closure.

This abstract model ignores the fact that all lines come from one coefficient field `c_xi`, but it shows that any proof must use that common-origin constraint explicitly.

## What the common-origin constraint would need to provide

The real line functions satisfy

```math
H_{a,b}(t)=c_{a+t(a+b)}c_{b-t(a+b)}.
```

Hence different pair-lines share many coefficients `c_xi`.  This shared vertex origin is the only possible source of forced triangle closure.

A successful phase-comparison lemma must therefore prove that incoherent labels cannot occur for line functions generated by one common coefficient field while still maintaining large shear energy.

Equivalently, it must rule out the following scenario:

```math
\text{many pair-lines have strong individual modes,}
```

but

```math
\mu(a,b)+\mu(b,o)+\mu(o,a)\ne0
```

for most anchored triangles.

## Why Cauchy--Schwarz does not immediately help

A Cauchy--Schwarz argument applied to the linewise shear sum can amplify the number of biased lines, but it still remains linewise unless it creates repeated shared vertices.

To force triangle closure, one needs an estimate involving products such as

```math
H_{a,b}(s)H_{b,o}(s)H_{o,a}(s),
```

or some physical-space equivalent that couples three pair-lines.  This is a genuinely higher-order statistic than the original shear autocorrelation.

## Consequence for the affine coboundary route

The affine/isotropic pair-lift route now has a conditional status.

It is viable only under one of the following additional inputs:

1. **Phase-comparison input.**  Large shear energy plus common-origin factorization forces many anchored triangle closures.
2. **Sifting input.**  A Kelley--Meka/Raghavan-style iterative sifting procedure selects a subcloud where line modes cohere into many anchored closures.
3. **Higher-order energy input.**  A separately controlled statistic supplies large anchored triangle averages.

Without such an input, the route stalls at individual line-mode bias.

## Branching interpretation

Failure of anchored triangle closure should not be treated as harmless noise.  It creates a branch:

```math
\text{strong individual line modes but incoherent triangle labels}.
```

Possible interpretations are:

1. the affine mode labels are the wrong variables;
2. non-isotropic quadratic chirps require richer labels including quadratic line coefficients;
3. the incoherence cancels when attempting global aggregation, requiring a sifting step to expose the surviving coherent part;
4. the incoherent labels concentrate on a low-rank/additive substructure.

## Updated proof architecture

The current phase-sensitive route should be rewritten as a trichotomy:

```math
\text{large shear contribution}
\Rightarrow
\begin{cases}
\text{affine coherent branch: many anchored triangle closures},\\
\text{non-isotropic chirp branch: extract quadratic line coefficients},\\
\text{incoherent branch: require sifting or expose concentration}.
\end{cases}
```

The affine coherent branch can proceed to coboundary cleaning.  The other two branches require new extraction lemmas.

## Immediate next proof task

Do not attempt to prove coboundary cleaning directly from line-mode extraction.

Instead, choose between two concrete next steps:

1. formulate a **sifting lemma** that upgrades many incoherent biased lines into a coherent subfamily with many anchored triangle closures; or
2. formulate the **non-isotropic chirp-label branch**, where each active line carries both a quadratic coefficient and an affine coefficient.

The second option may be more faithful to the quadratic model, because fixed-shift autocorrelation only sees affine modes strongly when the quadratic line coefficient is absent or suppressed.
