# Side-anchor deletion DAG

## Status

Exact physical-space reformulation of the side-anchor-sponsored sparse recursion.

The selected three-term progressions define an acyclic directed dependency graph on the parent block.  Every deleted side anchor points to the two progression points that survive it.  The terminal singleton step is the edge length of this affine dependency.

This identifies the remaining overlap as merging inside a geometric DAG, rather than arbitrary overlap among child states.

## Construction

Let

```math
D\subseteq[N,2N)
```

be four-term-progression-free.  Run the side-anchor deletion process:

1. choose a three-term progression in the current set;
2. delete its coordinated side anchor;
3. continue until the residual set is three-term-progression-free.

Write the selected progressions in deletion order as

```math
P_i=(a_i,b_i,c_i),
\qquad i=1,2,\ldots,K,
```

where:

- `a_i` is the deleted side anchor;
- `b_i` is the middle point;
- `c_i` is the opposite endpoint;
- the three points satisfy
  ```math
  a_i+c_i=2b_i.
  ```

The coordinated side role determines whether

```math
a_i<b_i<c_i
```

or

```math
a_i>b_i>c_i.
```

In either case the common difference is

```math
q_i=|b_i-a_i|=|c_i-b_i|.
```

## Time ordering

At the moment `a_i` is deleted, the points `b_i,c_i` remain in the current set.  Therefore each of them is either:

1. deleted at a later time; or
2. retained in the final three-term-progression-free residual set.

Let

```math
\tau(x)
```

be the deletion time of `x`, with `tau(x)=infinity` for residual vertices.  Then

```math
\boxed{
\tau(a_i)<\tau(b_i),
\qquad
\tau(a_i)<\tau(c_i).
}
```

## Directed dependency graph

Define a directed graph `Gamma` on vertex set `D` by adding

```math
a_i\longrightarrow b_i
```

and

```math
a_i\longrightarrow c_i
```

for every selected progression.

The time inequalities imply that every directed edge increases `tau`.  Hence

```math
\boxed{\Gamma\text{ is acyclic}.}
```

Every deleted vertex has outdegree exactly two.  Every residual vertex has outdegree zero.  Thus

```math
\boxed{
|E(\Gamma)|=2K,
\qquad
K\ge|D|-r_3(N).
}
```

The sink set is precisely the residual three-term-progression-free set and therefore has cardinality at most `r_3(N)`.

## Affine edge relation

The two outneighbors of a deleted vertex are not arbitrary.  They satisfy

```math
\boxed{
c_i=2b_i-a_i.}
```

Equivalently, the directed two-edge fork

```math
a_i\to b_i,
\qquad
a_i\to c_i
```

is an affine three-term progression.

The terminal singleton side child generated by this deletion is

```math
\{q_i\},
```

where

```math
q_i=|b_i-a_i|.
```

The selected middle membership, when present, is the same label `q_i` grouped at the center vertex `b_i`.

## Injective sponsor interpretation

Every selected progression has a unique deleted source `a_i`.  Therefore:

- terminal side memberships are indexed injectively by non-sink vertices;
- selected middle memberships are also indexed injectively by a subset of non-sink vertices;
- the middle lift associated with `q_i` contains the sponsor `a_i` as one of its two reflected points.

Thus every middle membership already carries one distinct physical parent point.  The unresolved physical-capacity gain must come from the remaining center and opposite-endpoint vertices, where many directed forks may merge.

## No repeated ordered fork

The ordered pair `(b_i,c_i)` determines the source uniquely:

```math
a_i=2b_i-c_i.
```

Hence two distinct deleted vertices cannot have the same ordered pair of outneighbors.

```math
\boxed{
(a_i\to b_i,c_i)
\ne
(a_j\to b_j,c_j)
\quad(i\ne j).
}
```

The DAG may nevertheless have substantial merging through one common outneighbor.

## Hypergraph degeneracy

The selected progression family also forms a three-uniform hypergraph

```math
\mathcal H=\{\{a_i,b_i,c_i\}:1\le i\le K\}.
```

The deletion order has the property that `a_i` belongs to no later selected edge.  Therefore every nonempty subfamily of `mathcal H` has an edge containing a vertex that appears in no later edge of that subfamily.

In particular, reversing the deletion order shows that each selected progression introduces at least one new point.  Consequently

```math
\left|\bigcup_{i\in I}P_i\right|
\ge
|I|+2
```

for every initial deletion family, with the obvious adjustment for a general subfamily after ordering it by deletion time.

This is an acyclic or shellable incidence structure, stronger than an arbitrary family of three-term progressions.

## Directed paths

A directed path

```math
x_0\to x_1\to\cdots\to x_h
```

records repeated reuse of progression points as later sponsors.

At each step, `x_{j+1}` is either the center or opposite endpoint of the progression sponsored by `x_j`.  Hence

```math
x_{j+1}=x_j+\sigma_j q_j
```

or

```math
x_{j+1}=x_j+2\sigma_j q_j,
```

where `sigma_j in {−1,+1}`.

All vertices remain in `[N,2N)`, while deletion times strictly increase.  Thus long directed paths are possible only through repeated affine cancellation among the step labels.

## Relation to the pure middle recursion

A selected middle child at center `b` consists of step labels `q_i` from incoming sponsored forks with

```math
b_i=b.
```

The side-anchor-sponsored recursion continues only through these grouped incoming labels.  Hence persistent middle nodes are precisely indegree fibers of the deletion DAG, resolved by the common difference label.

The next recursive generation applies the same construction to such a fiber of edge lengths.

Therefore the multiscale recursion alternates between:

1. an acyclic affine DAG on physical parent points;
2. a four-term-progression-free set of incoming edge lengths at a center.

## Current bottleneck

The sponsor injection pays for one physical point per selected progression.  To recover the full scale-compensated `4/3` capacity, one needs an additional point for approximately every three sponsored progressions.

In DAG language, the missing statement is a merging bound:

> A four-term-progression-free affine deletion DAG cannot merge the centers and opposite endpoints of too many sponsored forks without exporting a controlled family of smaller edge lengths.

The componentwise side-middle scale-export theorem proves this for the overlap generated by one paired side and middle state.  The unresolved task is to sum such local export across the whole deletion DAG without counting the same parent point or lower-scale edge more than a bounded number of times.

## Revised global target

Construct a potential on the deletion DAG satisfying

```math
\boxed{
\text{distinct sponsors}
+
\text{distinct lower-scale exported edge lengths}
\ge
\frac43K-O(r_3(N)).
}
```

A bound of this form, iterated on the indegree fibers of edge lengths, would match the coordinated harmonic branching constant and provide the desired telescoping mechanism.
