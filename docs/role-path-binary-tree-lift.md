# Role-path binary-tree lift

## Status

Exact physical-space lift theorem for the coordinated role-compressed recursion.

Every retained role child is built from one of three affine two-branch patterns around an anchor point.  Because the retained step set has pairwise-disjoint first three dilates, these two branches are disjoint.  Iterating along a depth-`h` path produces a full rooted binary affine tree inside the original set.

The exact size is

```math
\boxed{
2^h|D_h|+(2^h-1).
}
```

For a singleton terminal direction, a depth-`h` path therefore forces exactly `2^{h+1}-1` distinct root points.

This identifies the natural physical capacity base as `2`.  Since the coordinated harmonic branching lower bound is `8/3`, any proof closing the recursion must show that overlap among these binary affine trees cannot absorb the exponential gap.

## One role edge

Let `D` be a retained child step set anchored at a point `x` of its parent.

The three possible roles use the following non-anchor affine branches:

```math
A_1=\{1,2\},
```

```math
A_2=\{-1,1\},
```

and

```math
A_3=\{-2,-1\}.
```

Thus membership `s in D` implies that the parent contains

```math
x+a s
\qquad(a\in A_i)
```

as well as the anchor `x`.

Let `S subseteq D`.  Define its one-edge lift by

```math
\Lambda_{i,x}(S)
=
\{x\}
\cup
\bigcup_{a\in A_i}(x+aS).
```

## Exact one-edge cardinality

For every retained child `D`, the sets

```math
D,2D,3D
```

are pairwise disjoint.

### First role

For `i=1`,

```math
\Lambda_{1,x}(S)
=
\{x\}\cup(x+S)\cup(x+2S).
```

Since `S subseteq D` and `D cap 2D = emptyset`, the two translated branches are disjoint.  They are also disjoint from the anchor because `S` consists of positive integers.  Hence

```math
|\Lambda_{1,x}(S)|=1+2|S|.
```

### Middle role

For `i=2`,

```math
\Lambda_{2,x}(S)
=
\{x\}\cup(x-S)\cup(x+S).
```

The two branches are disjoint because `S` is positive, and neither contains `x`.  Thus

```math
|\Lambda_{2,x}(S)|=1+2|S|.
```

### Last role

For `i=3`,

```math
\Lambda_{3,x}(S)
=
\{x\}\cup(x-2S)\cup(x-S).
```

Again `S cap 2S = emptyset`, so the branches are disjoint, and positivity separates them from the anchor.  Therefore

```math
|\Lambda_{3,x}(S)|=1+2|S|.
```

In all cases,

```math
\boxed{
|\Lambda_{i,x}(S)|=1+2|S|.
}
```

## Depth-`h` path

Consider a coordinated role path

```math
D_0
\longrightarrow
D_1
\longrightarrow
\cdots
\longrightarrow
D_h,
```

where the edge from `D_{t-1}` to `D_t` has anchor `x_t` and role `i_t`.

Start with an arbitrary nonempty subset

```math
S_h\subseteq D_h.
```

Lift recursively by

```math
S_{t-1}
=
\Lambda_{i_t,x_t}(S_t)
\qquad(t=h,h-1,\dots,1).
```

By the definition of the role child,

```math
S_{t-1}\subseteq D_{t-1}.
```

The one-edge cardinality identity gives

```math
|S_{t-1}|=1+2|S_t|.
```

Solving the recurrence yields

```math
\boxed{
|S_0|
=
2^h|S_h|+(2^h-1).
}
```

Taking `S_h=D_h` gives

```math
\boxed{
2^h|D_h|+(2^h-1)
\le
|D_0|.
}
```

This is the exact path-capacity inequality.

## Singleton terminal direction

If

```math
S_h=\{d\},
```

then

```math
\boxed{
|S_0|=2^{h+1}-1.
}
```

Thus every depth-`h` path ending at one direction creates a full binary affine tree with:

- `2^h` terminal leaves;
- `2^h-1` internal anchor images;
- `2^{h+1}-1` total distinct root points.

The earlier binary-cube lift counted only the terminal leaves.  The role-star formulation also exposes all internal anchors.

## Explicit affine form

Choose at each level one coefficient

```math
a_t\in A_{i_t}.
```

A leaf of the lifted tree has the form

```math
x_1
+a_1x_2
+a_1a_2x_3
+\cdots
+\left(\prod_{r=1}^{h-1}a_r\right)x_h
+\left(\prod_{r=1}^{h}a_r\right)d.
```

Internal vertices are obtained by terminating this expression at an earlier anchor.

The exact cardinality theorem says that all vertices generated by the recursive two-branch construction are distinct along one path, despite possible sign changes and mixed roles.

## Depth bound

A nonempty depth-`h` path requires

```math
2^{h+1}-1\le|D_0|.
```

Therefore

```math
\boxed{
h\le\lfloor\log_2(|D_0|+1)\rfloor-1.
}
```

This is only a single-path bound.  It does not prevent many distinct paths from producing heavily overlapping binary trees.

## Relation to coordinated role multiplicity

Under coordinated valuation compression, every parent three-AP produces at most two retained child memberships.

The binary-tree lift shows that the physical-space capacity along one retained path also has base `2`.

Thus two distinct appearances of the number `2` are now separated:

1. **local role multiplicity:** at most two retained roles per parent three-AP;
2. **single-path physical capacity:** exactly two affine branches per level.

The harmonic mass lower bound remains

```math
\left(\frac83\right)^h
```

before overlap.  Hence the exponential gap is

```math
\left(\frac{4}{3}\right)^h
```

relative to the binary physical capacity of one path.

## Remaining obstruction

Different paths can share anchors, branches, and terminal leaves.  The depth-two finite-avoidance construction shows that arbitrary affine lifts may have large overlap, although it was not tailored to the coordinated valuation rule.

The next target is therefore a role-compatible tree-overlap theorem:

> Bound the weighted number of coordinated binary affine trees passing through the same root vertices, using the facts that every internal child is 4-AP-free, has pairwise-disjoint first three dilates, and is produced by a globally coordinated valuation rule.

An overlap rate strictly below `(4/3)^h` beyond the unavoidable binary capacity would contradict the `8/3` harmonic mass growth.
