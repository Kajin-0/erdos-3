# Forced-fork reserve: Bellman no-go and retained role

## Status

Exact finite no-go theorem for one proposed use of the parent-intrinsic forced-fork reserve.

The root-forced fork lemma gives a positive schedule-independent lower bound

```math
\sum_q H(\Xi_q^\sigma)\ge\Psi(S)
```

for every complete coordinated deletion schedule on the recorded states. This is genuine local structure. It does **not** make

```math
P\Psi(S)
```

a standalone stored Bellman potential.

The recorded factor-four transition

```math
S_1\longrightarrow S_2
```

already disproves that use for every nonnegative weight.

## 1. Bellman-compatible feature

The forced-fork quantity `Psi(S)` is harmonic mass in the lower-scale labels generated by one parent occurrence. With certified persistence multiplicity `P`, the direct Bellman-unit feature is

```math
F(S)=P\Psi(S).
```

A standalone nonnegative linear potential would have the form

```math
\Phi_\kappa(S)=\kappa F(S),
\qquad
\kappa\ge0.
```

For a one-child recorded transition without an error allowance, it would need

```math
D(S)+\Phi_\kappa(S')\le\Phi_\kappa(S),
```

or equivalently

```math
\kappa\bigl(F(S)-F(S')\bigr)\ge D(S).
```

## 2. Exact sign obstruction at `S_1 -> S_2`

The recorded step has scale factor `4`, so its positive Bellman debt magnitude is

```math
D(S_1)
=
\frac{P_1(3|S_1|+4)}{L_1}
\left(\frac84-1\right)
=
\boxed{\frac54}.
```

The forced-fork features are

```math
F(S_1)
=
2\cdot\frac1{21}
=
\frac2{21},
```

and

```math
F(S_2)
=
4\cdot\frac{388668}{6990295}
=
\frac{1554672}{6990295}.
```

Their parent-minus-child coefficient is

```math
\boxed{
F(S_1)-F(S_2)
=
-\frac{18667522}{146796195}
<0.
}
```

Therefore, for every

```math
\kappa\ge0,
```

one has

```math
\kappa\bigl(F(S_1)-F(S_2)\bigr)\le0
<
\frac54.
```

Hence

```math
\boxed{
\text{no nonnegative multiple of }P\Psi
\text{ is a standalone stored Bellman potential.}
}
```

The obstruction is stronger than insufficient magnitude: the coefficient has the wrong sign.

## 3. Unit transition credit is also insufficient

The forced-fork reserve can instead be viewed as a charge generated at a transition. With unit coefficient, compare the factor-four debt to the parent charge

```math
P_h\Psi(S_h).
```

For the recorded factor-four steps through `S_5`, the exact ratios are

```math
\frac{D(S_1)}{P_1\Psi(S_1)}
=
\frac{105}{8}
=13.125,
```

```math
\frac{D(S_3)}{P_3\Psi(S_3)}
=
\frac{15368877363127658643}
{1722549498740988928}
>8,
```

and

```math
\frac{D(S_4)}{P_4\Psi(S_4)}
=
\frac{221024462498552230659735994125}
{8318654923632839692153748992}
>26.
```

Thus the raw forced-fork harmonic charge does not by itself pay any recorded factor-four debt at unit weight.

This does not make the forced-fork theorem useless. It identifies unavoidable lower-scale structure that a stronger packing or obstruction theorem may amplify. It only rules out the simplest accounting interpretation.

## 4. Correct retained role

The forced-fork reserve should now be treated as one of the following:

1. a guaranteed transition output that feeds a separate packing theorem;
2. a lower bound on the novelty-plus-overlap dichotomy;
3. a seed for affine obstruction coverage or rectangle growth;
4. one coordinate in a larger potential containing target deficits, completion support, and overlap provenance.

It should not be described as an already telescoping reserve.

The active requirement is a theorem converting forced-fork occurrence mass into one of two durable forms:

```text
new obstruction support that survives to later scales
or
overlap charge that can be paid only a bounded number of times.
```

Without that conversion, the same forced labels may be reused across generations and the Bellman debt remains unpaid.

## 6. Scope

This result rules out only the standalone feature `P*Psi` with a nonnegative coefficient. It does not rule out:

- a transformed forced-fork deficit;
- a transition credit with a proved amplification constant;
- a nonlinear reserve;
- a combined rectangle/completion/overlap potential;
- a branching packing theorem;
- or the four-term Erdős conjecture.
