# Spectral line-mode obstruction

## Status

Proof-audit refinement.  This note decomposes the spectral pair-fiber autocorrelations along the one-dimensional orbits generated by each nonzero spectral sum `w`.

The purpose is to identify what kind of oscillation is forced when the pure four-balanced term is negative but no large linear Fourier coefficient is available.

## Setup

Let

```math
G=F_p^n,
\qquad p>4,
```

and let

```math
f=1_A-\alpha,
\qquad
c_\zeta=\widehat f(\zeta).
```

For each spectral sum `w`, define

```math
h_w(u)=c_u c_{w-u}.
```

The unipotent shear decomposition gives

```math
Q=\sum_w \langle h_w,\tau_{2w}h_w\rangle.
```

In the pure branch,

```math
\sum_{w\ne0}\langle h_w,\tau_{2w}h_w\rangle\le -c\alpha^4.
```

## Line decomposition inside each fiber

Fix `w\ne0`.  The shift

```math
u\mapsto u+2w
```

acts along affine lines parallel to `w` in frequency space.

Choose representatives `r` for the quotient

```math
G/\langle w\rangle.
```

For each representative `r`, define a function on `F_p` by

```math
H_{w,r}(t)=h_w(r+tw)
=c_{r+tw}c_{w-r-tw}.
```

Then

```math
\langle h_w,\tau_{2w}h_w\rangle
=\sum_{r\in G/\langle w\rangle}\sum_{t\in F_p}H_{w,r}(t)\overline{H_{w,r}(t+2)}.
```

## One-dimensional Fourier decomposition

Let

```math
\widehat H_{w,r}(m)=E_{t\in F_p}H_{w,r}(t)e_p(-mt)
```

with normalized one-dimensional Fourier transform.

Then, up to the normalization factor associated with summing instead of averaging over `t`,

```math
\sum_t H_{w,r}(t)\overline{H_{w,r}(t+2)}
=\sum_m |\widehat H_{w,r}(m)|^2 e_p(2m).
```

Taking real parts,

```math
Re\sum_t H_{w,r}(t)\overline{H_{w,r}(t+2)}
=\sum_m |\widehat H_{w,r}(m)|^2 \cos(4\pi m/p).
```

Thus negative spectral-pair autocorrelation requires mass in line modes `m` for which

```math
\cos(4\pi m/p)<0.
```

## Consequence

If all spectral pair-fibers were low-oscillation along their own `w`-lines, then the autocorrelations could not produce the negative pure obstruction.

Therefore the pure branch forces a substantial amount of line-mode oscillation in the products

```math
c_{r+tw}c_{w-r-tw}.
```

This is more structured than generic quadratic uniformity: it is oscillation of paired Fourier coefficients along lines whose direction is the spectral sum `w`.

## Relation to quadratic phase

A quadratic phase has Fourier mass whose phases change linearly along affine lines in frequency space.  Products of paired coefficients along

```math
u + (w-u)=w
```

can therefore have coherent nonzero line modes under translation by `w`.

This is consistent with the expectation that common extremizers of physical and spectral pair-fiber flatness should be quadratic rather than linear.

## Why this does not yet solve the problem

Line-mode oscillation alone does not imply a density increment.  One must still show that enough such oscillations across many `w` either:

1. concentrate on a low-rank quadratic factor;
2. produce an affine density increment of size `alpha^{2-epsilon}`; or
3. form a high-rank quadratic obstruction suitable for the relative-host branch.

## Candidate line-mode lemma

A possible next theorem is:

> Suppose
> ```math
> \sum_{w\ne0}\langle h_w,\tau_{2w}h_w\rangle\le -c\alpha^4.
> ```
> If no Fourier coefficient has size `alpha^{2-epsilon}` and the marginal fourth moment is small, then there is a nontrivial collection of pairs `(w,r)` such that `H_{w,r}` has large Fourier mass on nonzero one-dimensional modes.
>
> Moreover, the collection of such line modes either yields a low-rank quadratic factor or is high-rank enough to pass into the relative-host branch.

## Next research question

Can the line-mode data

```math
(w,r,m),
\qquad
|\widehat H_{w,r}(m)|\text{ large},
\qquad m\ne0,
```

be organized into an actual quadratic form or quadratic factor on `G`?  This is now the concrete bridge from spectral shear cancellation to quadratic structure.
