# Full-color coordinated branching

## Status

Exact strengthening of coordinated valuation compression.

The earlier coordinated theorem retained one global middle color and therefore kept only at least one third of the middle-role harmonic load. This loss is unnecessary if the three middle colors are emitted as separate children.

Each three-term progression still belongs to exactly:

1. one side child;
2. one middle-color child.

Thus the per-progression child multiplicity remains exactly two, while the total retained harmonic load increases from at least `4/3` to exactly `2` times the full weighted three-AP load.

---

## 1. Setup

Let

```math
B=A\cap[N,2N),
\qquad
|B|=\alpha N,
```

where `A` is four-AP-free.

For `x\in B`, define the three role-star step sets

```math
U_1(x)=\{s:x,x+s,x+2s\in B\},
```

```math
U_2(x)=\{s:x-s,x,x+s\in B\},
```

and

```math
U_3(x)=\{s:x-2s,x-s,x\in B\}.
```

Let

```math
\mathcal L(B)
=
\sum_{p,s:\ p+s,p+2s,p+3s\in B}
\frac1s
```

be the weighted three-AP load.

Every progression occurs once in each role, so

```math
\sum_xH(U_i(x))=\mathcal L(B)
\qquad(i=1,2,3).
```

The established deletion estimate gives

```math
\mathcal L(B)
\ge
2\left(
\alpha-
\frac{r_3(N)}N
\right).
```

Every `U_i(x)` is four-AP-free.

## 2. Coordinated side children

Let

```math
\epsilon(s)=v_2(s)\pmod2.
```

Define

```math
V_1(x)
=
\{s\in U_1(x):\epsilon(s)=0\},
```

and

```math
V_3(x)
=
\{s\in U_3(x):\epsilon(s)=1\}.
```

For each child, the sets

```math
V_i(x),
\qquad
2V_i(x),
\qquad
3V_i(x)
```

are pairwise disjoint.

Every three-AP of step `s` belongs to exactly one side child:

- even `v_2(s)`: its first-point child;
- odd `v_2(s)`: its last-point child.

Therefore

```math
\boxed{
\sum_xH(V_1(x))
+
\sum_xH(V_3(x))
=
\mathcal L(B).
}
```

## 3. All three middle colors

Define

```math
\chi(s)
=
v_2(s)-v_3(s)\pmod3.
```

For every color

```math
c\in\mathbb Z/3\mathbb Z,
```

set

```math
V_{2,c}(x)
=
\{s\in U_2(x):\chi(s)=c\}.
```

The three color classes partition `U_2(x)`:

```math
U_2(x)
=
V_{2,0}(x)
\sqcup
V_{2,1}(x)
\sqcup
V_{2,2}(x).
```

Each `V_{2,c}(x)` is four-AP-free because it is a subset of `U_2(x)`.

The inherited four-AP exclusions give

```math
V_{2,c}(x)\cap2V_{2,c}(x)=\varnothing
```

and

```math
V_{2,c}(x)\cap3V_{2,c}(x)=\varnothing.
```

It remains to exclude

```math
2V_{2,c}(x)\cap3V_{2,c}(x).
```

Suppose

```math
2a=3b
```

with `a,b\in U_2(x)`. Write

```math
a=3u,
\qquad
b=2u.
```

Then

```math
\chi(b)-\chi(a)
=
2\pmod3.
```

Hence `a` and `b` cannot have the same color. Therefore

```math
\boxed{
V_{2,c}(x),
2V_{2,c}(x),
3V_{2,c}(x)
\text{ are pairwise disjoint}
}
```

for every `x` and every color `c`.

Since the colors partition the full middle star,

```math
\boxed{
\sum_x\sum_{c\in\mathbb Z/3\mathbb Z}
H(V_{2,c}(x))
=
\mathcal L(B).
}
```

No maximum-color selection and no factor-three loss are needed.

## 4. Exact two-role branching theorem

Combining side and middle contributions gives

```math
\boxed{
\sum_x
\left(
H(V_1(x))
+
H(V_3(x))
+
\sum_cH(V_{2,c}(x))
\right)
=
2\mathcal L(B).
}
```

Every three-term progression contributes exactly twice:

1. once to its parity-selected side child;
2. once to the unique middle child determined by `\chi(s)`.

Therefore the child multiplicity per progression is exactly two, not merely at most two.

Using the weighted three-AP lower bound,

```math
\boxed{
\sum_{\rm children}H(C)
\ge
4\left(
\alpha-
\frac{r_3(N)}N
\right).
}
```

Since

```math
H(B)\le\alpha,
```

we obtain

```math
\boxed{
\sum_{\rm children}H(C)
\ge
4H(B)
-
4\frac{r_3(N)}N.
}
```

Thus the exact raw harmonic branching factor is `4`, with exactly two child incidences per weighted three-AP.

## 5. Scale descent and dyadic shelling

Every step in a three-AP contained in `[N,2N)` satisfies

```math
1\le s\le\frac N2.
```

Hence every child lies in `[1,N/2]` before dyadic shelling.

Dyadic shells are subsets of the role-color children. Therefore each shell remains:

1. four-AP-free;
2. equipped with pairwise-disjoint first three dilates.

The full-color construction is compatible with the same scale-descending recursion as the earlier coordinated theorem.

## 6. Why retaining all colors is legitimate

Emitting all three middle colors increases the number of nonempty child states indexed by `(x,c)`, but it does not increase the number of children generated by any individual progression.

A progression has one step `s`, hence one value of `\chi(s)`. It belongs to exactly one middle-color child.

Cross-child numerical overlap remains a packing problem, exactly as it already was for children with different anchors and roles. Discarding two middle colors does not solve that global overlap problem; it only discards valid harmonic production.

The correct local invariant is therefore:

```text
exactly two role-color memberships per progression,
not one globally chosen middle color per node.
```

## 8. Remaining whole-tree issue

The theorem is an occurrence-level branching identity. It does not by itself control:

1. overlap of different role-color children;
2. cross-generation reuse of the same numerical or pair token;
3. parent-external completion roots;
4. genuine ambient holes;
5. transport-target collision multiplicity.

Those are precisely the objects isolated by the affine pair-resource and sponsor-transport ledgers.

No generation-six computation is required to combine the two frameworks.
