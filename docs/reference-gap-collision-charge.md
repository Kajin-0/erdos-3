# Reference-gap collision charge

## Status

Symbolic multiscale transfer lemma for repeated oriented affine root
configurations after mandatory dyadic shelling.

The lemma converts unbounded branch multiplicity into a strictly lower-scale
four-AP-free reference-difference reserve. It does not yet bound reuse of that
reserve across different root configurations.

---

## 1. Repeated root configuration in one shell

Let `Q` be a finite root set and let `R` be a set of reference roots. Assume
that, for one fixed orientation `sigma`, every affine copy

```math
S_{r,\sigma}(Q)
=
\{\sigma(q-r):q\in Q\},
\qquad r\in R,
```

lies in one standard dyadic shell

```math
[M,2M).
```

Put

```math
a=\min Q,
\qquad
b=\max Q,
\qquad
\Delta=b-a.
```

For right orientation the shell conditions give

```math
b-2M<r\le a-M.
```

For left orientation they give the reflected interval

```math
b+M\le r<a+2M.
```

Therefore, in either orientation,

```math
\boxed{
\operatorname{diam}(R)<M-\Delta.
}
```

In particular, repeated references are possible only when `Delta<M`.

---

## 2. Reference-difference reserve

Let

```math
r_0=\min R
```

and define

```math
D(R)
=
\{r-r_0:r\in R,\ r>r_0\}.
```

Because `R` is a subset of the four-AP-free parent root set, translation and
subsetting give

```math
\boxed{D(R)\text{ is four-AP-free}.}
```

Every token in `D(R)` satisfies

```math
0<r-r_0<M-\Delta<M.
```

Thus `D(R)` is a strictly lower-scale reserve. After standard dyadic
resolution, every shell of `D(R)` lies below the repeated child shell base
`M`.

---

## 3. Harmonic multiplicity charge

Let

```math
m=|R|.
```

Since every token of `D(R)` is less than `M-Delta`,

```math
H(D(R))
=
\sum_{r\in R\setminus\{r_0\}}
\frac1{r-r_0}
>
\frac{m-1}{M-\Delta}.
```

Hence, for any physical pair `e={u,v}` contained in `Q`, with gap

```math
D_e=|u-v|,
```

the repeated occurrence mass of `e` satisfies

```math
\boxed{
\frac{m-1}{D_e}
<
\frac{M-\Delta}{D_e}
H(D(R)).
}
```

This is the basic reference-gap collision charge. It is state-independent and
uses no arithmetic-progression hypothesis beyond four-AP-freeness of the
ambient parent.

---

## 4. Three-AP edge-energy form

Suppose

```math
Q=\{x,x+d,x+2d\}
```

is a root three-AP. Then

```math
\Delta=2d
```

and its complete three-edge energy is

```math
J(Q)
=
\frac1d+\frac1d+\frac1{2d}
=
\frac5{2d}.
```

All `m` branches carry the same three active root pairs. Their excess over
first appearance is

```math
(m-1)J(Q).
```

The reference-gap charge gives

```math
\boxed{
(m-1)J(Q)
<
\frac52
\left(\frac Md-2\right)
H(D(R)).
}
```

The coefficient is the exact available reference slack measured in units of
the witness step.

When `d` approaches `M/2`, the reference interval collapses and reuse is
forced to vanish. Large reuse is possible only when the witness is short
relative to its shell.

---

## 5. Aspect-ratio decomposition

A sharper multiscale form keeps each reference separation individually. Fix a
pair gap `D_e` and partition the difference reserve into ratio bands

```math
D_k(R;e)
=
\{\delta\in D(R):
2^kD_e\le\delta<2^{k+1}D_e\},
\qquad k\in\mathbb Z.
```

For every nonempty band,

```math
\frac{|D_k(R;e)|}{D_e}
<
2^{k+1}H(D_k(R;e)).
```

Therefore

```math
\boxed{
\frac{m-1}{D_e}
<
\sum_k2^{k+1}H(D_k(R;e)).
}
```

This writes repeated pair mass as harmonic mass of lower-scale reference
differences weighted only by their dyadic aspect ratio `delta/D_e`.

---

## 6. Relation to collision fibers

For a fixed repeated root three-AP `Q`, the oriented completion maps force one
of the layer families

```math
2Q-R,
\qquad
2R-Q,
\qquad
\frac{R+Q}{2}.
```

Each is three equal translates of an affine image of `R`. The present lemma
extracts from that layer geometry the translated reference reserve `D(R)`.
Thus the collision debt has two simultaneous certificates:

```text
three-translate support in the parent;
strictly lower-scale reference-difference reserve.
```

The first controls geometry; the second supplies a harmonic transfer object.

---

## 7. Remaining reuse problem

The lemma closes one fixed-configuration collision fiber. A full treewise
inequality must still control two forms of reuse:

1. the same reference-difference token may be generated by several distinct
   repeated root configurations;
2. one physical pair may belong to several witness configurations inside one
   child shell.

The approved next object is therefore a first-appearance ledger for
aspect-ratio tokens

```math
(\delta,D_e,k),
\qquad
2^kD_e\le\delta<2^{k+1}D_e,
```

combined with the existing affine physical-pair identity. No constant branch
multiplicity assumption is required.
