## 2. Universal pathwise coordinate

The correct triangular coordinate is the physical pair gap.

For an occurrence-owned pair measure with dyadic gap moment

```math
\Phi_p=\sum_e\mu(e)G(e)^p,
```

direct discharge is nonexpanding for every `p>=0`. Every mechanism except adjacent edge swap and multiplicity-one light support contracts or terminates.

The two equality mechanisms are finite:

```text

  involution; at most one new identity before recreation;

exact-gap multiplicity-one light support:
  at most nine nonterminal identities,
  or ten including a final local sink.
```

Therefore every first-appearance pair lineage reaches a terminal sink or recreates an earlier pair identity in finite time.

## 6. Active universal frontier

The local and pathwise transport problems are closed. The unresolved theorem is global branching and recreation accounting.

The next proof should construct a production-owned potential that controls:

1. the number or total mass of finite pair lineages created by one parent;
2. recreation of the same physical pair under different owners;
3. recreation of terminal affine sinks;
4. the exact `7/4` first-appearance depth release;
5. lower-gap release from recursive heavy transitions;
6. the relation between pair-lineage value and the original reciprocal mass.

The target is not another path rank or physical Hall inequality. It is a whole-tree occurrence-owned Bellman inequality of the schematic form

```math
\boxed{
\text{child lineage value}
+
\text{terminal value}
+
\text{released depth}
\le
\text{parent production-owned value}.
}
```
